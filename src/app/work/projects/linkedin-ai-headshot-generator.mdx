## Overview

Built an AI-powered **Telegram bot** that **creates professional LinkedIn headshots** using **Stable Diffusion for image generation** and **InsightFace for face swapping**.

### Key Highlights

- **AI-powered face swapping** for realistic profile photos.
- **Stable Diffusion** for high-quality **AI-generated images**.
- Integrated into **Telegram for instant use**.
